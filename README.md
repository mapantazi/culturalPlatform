# culturalPlatform
Web platform  designed to promote and visualize the cultural activities. Built with React,Tailwind CSS,Python data pipeline automatically converts Google Form into structured JSON data.
Project report, which includes a detail system evaluation along with proposed solutions and future enhancements.

This project is part of the "Craft Understanding, Education, Training, and Preservation for Posterity and Prosperity" Project ([craeft.eu](https://www.craeft.eu/)) - a European Union's Horizon Europe research and innovation programme under grant agreement No 101094349, the project focused on understanding, teaching and promoting traditional craftsmanship as living cultural heritage.

#Dependencies -> npm install
#Run Server -> npm start

#Data Pipeline
Google Form -> export on csv -> run python screipt ( python csv_to_json.py ) -> convert to JSON
